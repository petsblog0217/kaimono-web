# かいものメモ（Web版）

自分用・家族用の買い物メモWebアプリ（単一HTML・ビルド不要）。
パステル基調・線アイコン、お店ごとの買い物リスト＋家の在庫＋商品辞書。
家族とのリアルタイム共有は Firebase Realtime Database（合言葉=ルームコード方式）。

- 本体：`index.html`（1ファイル完結）
- 配信：GitHub Pages（このリポジトリ）
- 公開URL：https://petsblog0217.github.io/kaimono-web/

iOSアプリ版（SwiftUI）は別リポジトリ `magnet-memo`（非公開）。Web版はそこから派生した作り直し。
